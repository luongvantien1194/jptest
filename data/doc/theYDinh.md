# TÀI LIỆU CHUYÊN SÂU: THỂ Ý ĐỊNH (VOLITIONAL FORM - V-YO / V-OU)

Thể Ý định (意向形 - Ikoukei) dùng để rủ rê, đề nghị hoặc diễn tả một dự định mà người nói đang ấp ủ.

---

## 1. CÁCH CHIA ĐỘNG TỪ (CONJUGATION)

Quy tắc chia thể này tập trung vào việc biến đổi đuôi hàng **u** sang hàng **o**.

| Nhóm | Quy tắc biến đổi | Ví dụ (Từ điển → Thể Ý định) |
|:---:|:---|:---|
| **Nhóm 1** | Chuyển đuôi hàng **u** → **hàng o** + **う** | 書く (Kaku) → **書こう** (Kakou) |
| | | 買う (Kau) → **買おう** (Kaou) |
| | | 待つ (Matsu) → **待とう** (Matou) |
| **Nhóm 2** | Bỏ **る** + **よう** | 食べる → **食べよう** (Tabeyou) |
| | | 見る → **見よう** (Miyou) |
| **Nhóm 3** | **Bất quy tắc** | する → **しよう** (Shiyou) |
| | | 来る (Kuru) → **こよう** (Koyou) |

---

## 2. CÁCH DÙNG CƠ BẢN

### A. Rủ rê, đề nghị (Thân mật)
Thay vì dùng `~ましょう` (Mashou) lịch sự, ta dùng thể ý định với bạn bè, người thân.
- **Lịch sự:** 行きましょう (Ikimashou - Cùng đi nào!)
- **Thân mật:** **行こう** (Ikou - Đi thôi!)

### B. Tự nói với bản thân
Dùng khi người nói đang tự nhủ hoặc quyết định làm một việc gì đó ngay lúc đó.
- *Ví dụ:* さあ, **帰ろう**。 (Nào, về thôi nào.)

---

## 3. CẤU TRÚC DỰ ĐỊNH (PHỔ BIẾN TRONG NGỮ PHÁP)

Để diễn tả một dự định đã có từ trước hoặc đang ấp ủ, người ta thường dùng:
**[V-yo/ou] + と思っています (To omotte imasu)**

- **Ví dụ:**
  - 日本へ**行こう**と思っています。 (Tôi đang dự định đi Nhật.)
  - 新しい車を**買おう**と思っています。 (Tôi đang dự định mua xe mới.)

---

## 4. LOGIC LẬP TRÌNH (DEVELOPER NOTES)

1. **Trường hợp Nhóm 1 đuôi "u":** Lưu ý khi chuyển sang hàng "o" sẽ có các cặp như:
   - `u` -> `ou` (kau -> kaou)
   - `tsu` -> `tou` (matsu -> matou)
   - `ru` -> `rou` (kaeru -> kaerou)
2. **Trường hợp Nhóm 3:** Động từ `来る` (Kuru) có sự biến đổi âm đọc từ `Ku` thành `Ko` (**こよう**), cần đặc biệt lưu ý khi xử lý chuỗi ký tự (string manipulation).
3. **Mối quan hệ với Thể Lịch sự:** - V-yo/ou tương đương với `V-masu` bỏ `masu` + `mashou`.
