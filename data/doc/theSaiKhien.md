# TÀI LIỆU CHUYÊN SÂU: THỂ SAI KHIẾN (CAUSATIVE FORM)

Dùng khi chủ thể gây ra hành động cho một đối tượng khác (Bắt làm / Cho phép làm).

---

## 1. CÁCH CHIA ĐỘNG TỪ

Tất cả động từ sau khi chia sai khiến sẽ trở thành **Động từ Nhóm 2**.

| Nhóm | Quy tắc biến đổi | Ví dụ (Từ điển → Sai khiến) |
|:---:|:---|:---|
| **Nhóm 1** | Chuyển đuôi hàng **u** → **hàng a** + **せる** | 読む → **読ませる** (Bắt đọc) |
| | *Lưu ý đuôi う:* Chuyển thành **わ** | 買う → **買わせる** (Cho mua) |
| **Nhóm 2** | Bỏ **る** + **させる** | 食べる → **食べさせる** (Bắt ăn) |
| **Nhóm 3** | **Bất quy tắc** | する → **させる** |
| | | 来る → **こさせる** |

---

## 2. CẤU TRÚC TRỢ TỪ
1. **Tự động từ:** [Người A] は [Người B] **を** [V-sai khiến]. (Ví dụ: Bắt con đi học).
2. **Tha động từ:** [Người A] は [Người B] **に** [Vật] を [V-sai khiến]. (Ví dụ: Bắt con ăn rau).
